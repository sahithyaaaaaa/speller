# 📖 Speller
A C-based spell checker that efficiently verifies the spelling of words using a hash table. The program loads a dictionary into memory, checks words from a text file against the dictionary, and frees allocated memory upon completion. This project demonstrates the implementation of efficient data structures, file handling, and dynamic memory management.

## ✨ Features
- Loads a dictionary into memory using a hash table
- Performs fast and efficient word lookups
- Identifies misspelled words in a text file
- Uses dynamic memory allocation
- Properly frees allocated memory to prevent memory leaks

## 🛠️ Technologies Used
- C
- Hash Tables
- Linked Lists
- Dynamic Memory Allocation
- File I/O

## 🚀 Future Enhancements

- Spell correction suggestions
- Case-insensitive dictionary expansion
